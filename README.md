# fingec-assets

Ce dépôt est dédié au stockage et à la gestion des **assets** (images, icônes, polices, etc.) utilisés par les projets Fingec.

> Je fais ce repo pour les assets.

## Contenu

- `images/` : illustrations, logos et visuels
- `icons/` : icônes SVG et PNG
- `fonts/` : polices de caractères

## Utilisation

1. Clonez le dépôt :

```bash
git clone https://github.com/fingec/fingec-assets.git
```

2. Ajoutez ou mettez à jour les assets dans les dossiers appropriés.

3. Faites un commit et poussez vos changements.
